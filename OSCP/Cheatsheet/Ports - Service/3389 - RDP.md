# 基本操作

- RDP接続
	- ローカルでRemote Desktop UsersもしくはAdministratorsグループに所属しているユーザーが利用可能
	- ドメインポリシーによっては、上記のグループに所属していなくても使える
```zsh
# ADの場合、/d:<domain>を追加
# 接続ができない場合、/network:wan /audio-mode:1 /timeout:50000 を追加するか、sudo ifconfig tun0 mtu 1250
xfreerdp3 /dynamic-resolution +clipboard /cert:ignore /v:$TargetIP /u:<username> /p:'<pw>'
```

- RDPの有効化
	- 条件：ローカル管理者権限
	- （WinRMで十分なため、必要性は低い）
```powershell
# RDPを有効化してファイアウォールルールも設定
Set-ItemProperty -Path 'HKLM:\System\CurrentControlSet\Control\Terminal Server' -Name 'fDenyTSConnections' -Value 0
Enable-NetFirewallRule -DisplayGroup 'Remote Desktop'

# 確認→0なら成功
Get-ItemProperty -Path 'HKLM:\System\CurrentControlSet\Control\Terminal Server' -Name 'fDenyTSConnections'
```
- →RDP接続（`/d`は不要）
